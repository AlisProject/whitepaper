# 1. 什么是ALIS

ALIS是日本第一个分散型社交媒体平台。是使培养和发现出为人们创造出有价值内容的作者成为可能的全新社交媒体。使用ALIS虚拟货币作为核心的驱动力，未来可以实现广告和隐形营销软文的过滤，以及快速的发掘高品质的内容和值得信赖的人。准确而言，我们关于这个平台的构想极大的受启发于STEEM（https://steem.io ）。关于STEEM的介绍和说明可以参考这篇文章（https://cointelegraph.com/news/steemit-new-social-media-platform-which-pays-you-to-post ），由于这边文章非常易于理解，还希望您一定过目。我们的故事开始于我们的团队成员Sota Ishii将旅行中的几张照片发布在了STEEM（https://steemit.com/japanese/@sot528/1-south-america-tour-no-1 ）。那篇投稿在一天中获得了奖金$30的评价报酬，我们使用这份报酬，享受了Domino最高级的披萨。

我们怀着强烈的好奇心开始思考这种模式是如何得以实现，并开始彻底的调研STEEM（我们的动机当然不是为了再能够享受一次Dominno的顶级披萨）。随着对于STEEM的深入研究，我们越来越了解其优秀之处。STEEM通过平台本身的口碑来提升STEEM的虚拟货币的价值，*在交易所得到了很高的估值*。通过平台本身价值的提升，将升值的价值作为平台内容的作者以及评价者的回馈机制得以实现。另外，STEEM已经获得了$3.78亿（2017/7/9）的估值。回避了传统媒体将“广告”作为大部分收益（正如您所了解的，无论是Google，还是Facebook，七成以上的收益来源于广告），让用户在浏览各种内容的同时切入广告内容，无论如何也没有办法避免没有广告的浏览环境。对于这种境况，至今为止，广告界采取了各种各样的回避手段。例如，当用户浏览广告的时候积极的采取一些激励手段，或者近几年通过大数据分析和机器学习，针对不同的用户提供针对性的广告内容等。然而，迄今为止无论是何种方式，都没有取得显著的成效，并没有改变用户不想看到的广告，被近几年流行的营销软文误导消费的立场并没有得到改变。针对这种状况，STEEM给出的解决方式是改变根本利益规则。然而，我们认为STEEM有两点不足之处（即使有不足之处STEEM是一个非常成功的平台，并不改变我们从中获得了无限灵感）。第一是，平台的虚拟货币支付机制过于复杂导致用户需要花一些时间去理解。单就STEEM而言，存在STEEM SP和STEEM Dollars两种虚拟货币，获得方式也比较复杂，平台的运营规则对于初次使用的用户而言非常难于理解。这足以成为让低转化率的新用户望而却步的理由。最为媒体平台，我们认为最重要的是大幅提升Activation和Retention。第二点，完全不支持日语。我们为了创建一个解决了这些问题的新平台，创立了ALIS项目。ALIS具备至今社交媒体所没有的三个特点。

1. 可以更快的获取到高品质的内容
2. 将用户提供的价值回馈给用户
3. 通过区块链技术用低成本实现数据的高可信赖性

# 2. 我们的目标市场规模和日本市场的魅力

我们的目标市场预计2017年SNS（社交网络服务）：5.6兆日元，网络媒体：17兆日元，合计：22.6兆日元的市场。再加上这个市场从2014年开始处在高速增长（整体增长率为47%，其中SNS的增长率为130%）的阶段，这是个前景非常广阔的市场

其中，作为第一步，我们打算先从日本市场着手。首先，欧美国家在区块链技术上具有领先优势，而在日本基本不存在这种技术的运用企业。加之由于日本的《改正资金决济法》，海外区块链技术公司无法在日本国内经营，国内的相关公司也需要面对工商登记的巨大障碍。		
**これは一度日本において先行者としてのポジションを確立すれば、言語という壁が逆に強固なものになり独占的プレイヤーとしての立ち位置を確立することできる可能性を示唆している。**关于语言的障碍，像微软公司已经发布了同声翻译工具，预想未来五年世界，也许语言不再成为使用网络服务的障碍。加上日本以名义GDP是49390亿美元居于世界第三位，号称对外纯资产世界第一等而言，可以说是市场前景非常广阔。同时，日本的SNS和网络媒体市场规模最少估计也有1.1兆日元。一方面，这个市场由于日语的语言障碍，没有能够及时跟进国际平台（例如WhateApp，Tumblr，Pinterest等即时在日本即时能够使用也基本没有知名度）。但是正因如此，我们了解到日本市场应该被高度重视。从日本看来，在尚未被开拓的市场去的压倒性的份额，之后再进军国际市场，对于保证我们事业的高成功率而言，是一种合理的竞争策略。加上我们的主要成员都是日本的本土居民，熟知甚至精通日本的文化和商务规则，从而将日本市场作为起点对我们而言是最优方案。

# 6. 为什么可以收集到高品质的内容

让人们发布优质的内容，或者针对其他人的发布内容评价所必须的驱动力是什么？是对发布更过优质内容的作者或者对内容更多内容公正评价的评价者给予更多的报酬奖励。或者能够为他人做出贡献，满足自己被认可的想法等作为人的基本需求也可以认为是动机。设计能够切合这种需求的社交媒体平台是我们的首要任务。ALIS的报酬系统正式针对这种需求赋予项有力的动机。对于作者，自己的文章通过被更多的用户，或者拥有更多ALIS代币的用户评价获得相应的报酬称为可能。针对给予高质量评价的用户而言，对于评价人数越多的文章，评价时间越早就会得到越多的报酬。除此之外，拥有越多ALIS代币的用户越越能够获得更多的ALIS代币。总而言之，拥有更多的ALIS代币意味着创造优质内容或者发掘优质内容能够获得更多的奖励，进而能够进一步增加优秀内容的比率，从而创造一种良性循环。这就是为什么可以ALIS能够收集到高品质的内容。将这种模型以下图所示AARRR为基础的系统图进行分析可以得到同样的结果，可以看出奖励体系是平台成长的关键驱动力。以上虽然仅仅是着眼于奖励的物质面，但是这种奖励同于人获得认可的需求也是密不可分。明示获得奖励的数值，目的是可以直观了解用户对于他人或者平台的贡献都。即使不考虑ALIS代币的交换价值，仅仅是考虑其对用户的被认可价值，类似于Facebook的“赞！xx个”，也许可以期待其可以达到类似的效果。



# 7. 为什么可以给人们回馈价值

我们的代币本身不具有价值。但是随着ALIS平台价值得到认可，ALIS代币就会拥有相应的价值，可以在交易所通过高的价格进行交易。也许您对这一结论的初始印象会抱有怀疑的态度，但是通过同样方案的STEEM已经获得了$3.78亿的估值的事实来看这种方案是具有可行的。此外，通过STEEM的俄语版GOLOS同样也得到了$10亿以上的估值（2017/07/17）的事实来看，这种方案取得成功的可能性非常高。



# 8. 为什么ALIS可以持续的长期发展 	

ALIS平台的价值能够得到认可有一个最重要的要素，那就是平台具有用户粘性，为用户提供具有依赖性的服务。为此，有各种个方法，其中之一是执着于提升代币的性质。我们的代币在平台上有效（本章后续会说明）状态下具有50%的通货膨胀率，这一部分价值奖奖励给对于ALIS平台作出贡献的用户。这是近年来虚拟货币的目的被限定在交易上，为了规避这种风险而设定的数字。拥有ALIS代币，同时又能够为平台的发展作出努力的用户越多，就需要越多的代币这种机制的重要性为基础，设定了50%的通货膨胀率。但是如果仅仅这一个条件，就没有办法预防用户在ALIS代币升值到预期值之后马上进行交易兑换行为。对此，我们根据NEM的PoI（http://nemmanual.net/NEM_Technical_reference_JA/PoI/7_PoI.html ）引入以下代币于时间的逻辑关系式：

```
f(t) = log94(t) (1 <= t < 94)
f(t) = 1 (t >= 1)
```

其中 t 是代币在账户中存储的天数。采用这种逻辑的理由有三点。第一，为了让新用户尽早体验ALIS的魅力，t 越小，代币实效性增长速度就越高。第二，长期用户可以获得ALIS代币的100%价值。第三，每次取出之后，t 从新归零，从而避免了用户可以随意交易行为。真真想为ALIS贡献的用户数越多，这个逻辑就越具有合理性，可以确保平台长期有效的发展。



# 9. 代币是如何生成和发布的

ALIS预定通过ICO筹集资金，初期发行5亿枚，预定将其中的一半交换为Ethereum虚拟货币。发行上限为2.5亿枚，剩下的2.5亿由我们和我们的利益相关方保有。我们保有2.5亿也就是整体上限的50%的理由是，我们能够掌控维持平台健全发展的奖励机制。

但是请注意，当我们拥有最多的代币保有量并不意味着我们拥有创造平台价值的决定权。加上我们不会随意的交易所拥有的代币（我们会通过智能控制系统限制代币的随意交易，具体详情会在今后公布）

那么，之前有提到通过ICO发布的ALIS代币具有50%的通货膨胀率，那么关于通货膨胀的部分的价值如何分配，通过以下2点进行说明。

1. 对被公认发布优秀文章的作者进行奖励
2. 对多数人公认的优秀的文章尽早作出评价的用户进行奖励

对于这种奖励机制，ALIS代币的保有量越多，所获得的奖励也会越多。也就是说，长期对于ALIS平台作出贡献，或者ALIS代币保有量多的人们是我们最重要的利益相关群体，让这些人获得更多的利益是被作为规则。这个和PoI机制类似，如果平台**プラットフォームが壊れてしまうと困る重要度の高いユーザであれば、プラットフォームへの健全な貢献を心がけるはずであるという前提に則っている**。此外，如果平台上出现保有大量的ALIS代币的垄断型的用户，就有可能会左右平台的价值以迎合自己的意愿。

针对上述的风险，正如前面所述，ALIS代币在平台上的价值从冷却到有效需要一定的时间。防止保有大量代币用户在短时间内进行大量交易的同时，有能够让代币在平台上被持续保持，可谓一举两得。

首先先介绍ALIS代币的全体发行量。正如前面提到的ALIS代币的预定发行数量的上限为5亿枚，其中最高2.5亿将在Crowdsale销售。我们的团队将回收销售剩余的代币，并在3年间将其冻结。在通过Crowdsale发布ALIS代币中，假设放入exchange wallet中的比例为X1，放入ALIS wallet中的比例为X2，对于X2引入50%的通货膨胀率。这部分通过通货膨胀获得的代币将通过奖励机制分配给投稿作者和评价者。在详细介绍分配逻辑之前，首先先介绍我们的基本原则。

1. 文章的投稿作者和评价者都应该被得到重视和尊重，但是因为作者相对会付出更多的劳动，所以会加重针对作者的奖励份额
2. 文章如果获得越多的点赞，作者就可以获得越多的代币奖励，同样，评论如果获得越多的点赞数，评论着也可以获得越多的代币。
3. 代币的奖励量，奖励逻辑会随着平台的发展不断进化改善。

通过上述的原则，我们会将ALIS代币中的90%分配给作者，10%分配给评价者(假设达成我们ICO最低集资额3.5亿的情况下，在第一年对于作者6700万，评价者750万进行奖励分配。注意，这个假设ALIS价值不发生任何改变的情况)

这是由于初期文章的数量非常重要。但是，随着投稿文章数量的逐渐上升，文章评价的重要度也会逐渐上身。在这个阶段针对对于评价者的分配量应该会有所提高。这就需要在平台发展的不同阶段进行调整的系数，如果针对这些系数由我们运营者实施，就会产生中央集权机制。所以，当用户希望通过交涉参与的方式进行系数调整的场合，会进行以ALIS代币所有量作为权重的投票。根据总体意愿(51%以上同意)调整相关系数的运营方针。

接下来介绍对于每个作者和评价者的具体代币奖励方法。首先有一个基本点数Ai。这个基本点数是通过作者，评价者行为被计算出。保有越多ALIS代币的作者投稿的文章被设定的基准点数Ai越多，具体的表达式如下：

```
Ai = Valid ALIS tokens owned by user /
ALL valis ALIS tokens (Ai = 0.01 if Ai < 0.01)
```

拥有越多ALIS代币的用户就越有保护平台健全的动机，




# 11. 如何防止不正当操作 

根据上述的分配逻辑，如果允许以下不正当的方式取得代币，就有可能损害大多数用户的利益。

1. 通过同时注册做个账号，对自己的文章进行大量评价来获得ALIS代币奖励。
2. 通过和特定的用户合作，对制定的文章进行大量评价来获得ALIS代币奖励。


我们已经准备了防止这些不正当行为的对策。首先针对低一点，账号无法被轻易的注册。具体而言就是通过短信本人确认，或者必须通过Facebook认证登陆的方式进行抑制。针对第二点，

1. 当特定用户在短时间内评价大量文章时进行无效化设定（因为在短时间内能够同时阅读大量文章的用户，统计学上应该非常少）
2. 通过文章的Bad评价的比例来分析给予Bad评价用户的行为，如果确认为不正行为，将给予其奖励无效化。

等对策来处理不正行为。这受启发于STEEM已经采用的不正行为防止规则。这在STEEM平台上能够被有效运用，相信在我们的平台上也会同样有效。
