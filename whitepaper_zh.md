# 1. 什么是ALIS

ALIS是日本第一个分散型社交媒体平台。是能够使培养和发现为人们创造出有价值内容的作者的全新社交媒体平台。使用ALIS代币作为核心的驱动力，实现广告和隐形营销软文的过滤，以及快速的发掘高品质的内容和值得信赖的人。坦率而言，关于这个平台的构想极大的受启发于STEEM（https://steem.io）。关于STEEM的介绍和说明可以参考这篇文章（https://cointelegraph.com/news/steemit-new-social-media-platform-which-pays-you-to-post ），这边文章非常易于理解，还希望您一定过目。故事开始于我们的团队成员 Sota Ishii 将旅行中的几张照片发布在了STEEM（https://steemit.com/japanese/@sot528/1-south-america-tour-no-1 ）之后，那些照片在一天中获得了接近$30的评价报酬，我们使用这份报酬，享受了Domino最高级的披萨。

我们怀着强烈的好奇心开始研究这种模式是如何得以实现，并开始彻底的调研STEEM（我们的动机当然不是为了能够再享受一次 Dominno 的顶级披萨）。随着对于STEEM的深入研究，我们越来越了解其优秀之处。STEEM通过平台本身的口碑来提升STEEM代币的价值，并在 exchange 获得了很高的估值。STEEM实现了平台本身价值的提升的同时，将升值的价值回馈给平台内容的作者以及评价者的这种。STEEM已经获得了$3.78亿（2017/7/9）的估值。其回避了传统媒体将广告作为大部分收益（正如您所了解的，无论是Google，还是Facebook，七成以上的收益来源于广告），在用户在浏览各种内容时切入广告内容，使得用户无法避免充斥着广告的浏览环境。对于这种境况，至今为止，业界采取了各种各样的改善用户体验的手段。例如，当用户浏览广告的时积极的给予一定的激励，或者开始于近几年的通过大数据分析和机器学习，针对不同的用户提供针对性的广告内容等。然而，迄今为止无论是何种方式，都没能取得显著的成效。用户并没有能够摆脱不想看到的广告，容易被同时近几年开始流行的营销软文误导消费的状况并没有得到改善。针对这种情况，STEEM给出的解决方式是改变根本利益规则。但是，我们认为STEEM有两点不足之处（即使有所不足，STEEM仍然是一个非常成功的平台，并使我们从中获得了无限灵感）。第一，平台的代币机制过于复杂，用户需要花很多时间去理解。单就STEEM而言，存在STEEM SP和STEEM Dollars两种代币，获得方式也比较复杂，平台的运营规则对于初次使用的用户而言非常难于理解。这足以成为没有使用粘性的新用户望而却步的理由。作为媒体平台关键，我们认为最重要的是大幅提升用户活跃度和转化率。第二，STEEM完全不支持日语。我们认为需要一个新的平台来解决这些问题，从而创立了ALIS项目。ALIS具备至今为止社交媒体所没有的三个特点。

1. 可以更快的获取高品质的内容
2. 将用户提供的价值回馈给用户
3. 通过区块链技术用低成本实现数据的高可信度

# 2. 我们的目标市场规模和日本市场的魅力

我们的目标市场预计规模为：2017年SNS（社交网络服务）：5.6兆日元，网络媒体：17兆日元，合计：22.6兆日元。加之这个市场从2014年开始一直处在高速增长（整体增长率为47%，其中SNS的增长率为130%）的状态，所以可以说这是个前景非常广阔的市场。

作为第一步，我们打算先从日本市场着手。由于欧美国家在区块链技术上具有领先优势，而在日本基本不存在成熟运用这种技术的团体和机构。此外，日本的《改正资金决济法》规定，海外的类似团体和机构无法在日本国内直接运营，国内的相关团体如果希望运营企业，也需要跨过工商登记的巨大障碍。如果一旦能够确立在日本先行者的地位，语言障碍反而会成为优势，让ALIS成为日本区块链技术运用的独家公司成为可能。针对使用平台的语言的障碍的难题，类似微软公司已所发布的实时翻译工具，预想未来五年世界，也许语言障碍不会再困扰用户使用国外的网络服务。日本以名义GDP为49390亿美元居于世界第三位，并号称对外纯资产世界第一等，可以说拥有非常广阔的市场前景。同时，日本的SNS和网络媒体市场规模最少估计也有1.1兆日元。一方面，这个市场由于日语的语言障碍，没有能够及时跟进国际平台（例如WhateApp，Tumblr，Pinterest 等虽然在日本也能够使用，但基本缺乏知名度）。正因如此，我们看到日本市场应该被高度重视。对于日本而言，在尚未被开拓的市场更容易取得压倒性的份额，之后再进军国际市场，对于保证我们事业的成功率而言，是非常合理的竞争策略。加上我们的主要成员都是日本的本土居民，熟知甚至精通日本的文化和商务规则，从而将日本市场作为起点对我们而言是最优之选。



# 3. 面向日本的平台成长战略

作为针对日本市场的服务成长战略，我们预想了三个阶段。

1. 在独特的领域中确立市场份额（虚拟货币，未成年人动漫等）
2. 在所有点评网站领域拓展市场（饮食，旅行，减肥瘦身等日常领域到入学，结婚，购房等生命活动领域）
3. 以积累的用户信用度信息为基础，开拓新的领域

首先对第1阶段进行说明。ALIS的建立者 Masahiro Yasu 拥有日本SNS（日本版Linkedin）的开发经验。通过这次经验，理解到社交平台成长最重要的要素之一是“来自用户的反馈”。其重要性对于作为Facebook，Instagram等SNS的用户自然可以理解，通过了解大多数用户是否关心自己投稿内容的反馈这一点就很明确了。另外，作为AARRR语境中初期阶段，最重要的是降低用户的流失率，这其中最重要的课题之一就是用户反馈。幸运的是，这个课题可以通过虚拟货币分配逻辑来实现和解决，也就是包含报酬体系的ALIS特有解决方案，这其中也蕴含了巧妙的经济规则。服务成长的第二个要素，也是FaceBook和曾经在日本获得成长的mixi等SNS的共通条件之一：“从内部服务开始”。Facebook作为哈弗大学的社交工具，mixi作为日本大学的工具都是从“内部”服务开始。针对这一点，首先将“对于虚拟货币有兴趣的人”作为目标群体，更容易增加平台的投稿量。作为前期战略，理由很也就很明确了。

1. 关于虚拟货币相关的信息，网上充斥着低信赖度的信息，人们期望获得更可靠信息源。
2. 关于虚拟货币的情报，谁都可以发布误导信息（机会主义的技术分析被发现，XX要上涨了！看到这种twitter被误导购买之后，最终后悔的人有多少人呢？请放心我也是其中一员）
3. 对于我们ALIS的成员而言没有什么比虚拟货币·代币更美妙，更值得充满热情的去报道和分享相关的信息的了

基于上述的成长战略中的1的概要，可以总结为以下两点。

1. 热衷于独特的领域 + 高信赖度信息的用户数量较多的领域（初期预想是虚拟货币领域），集客将事半功倍。
2. 通过报酬奖励机制保证用户在投稿之后可以获得较多的反馈（通过指定标签获得新记事的通知，对于自己投稿留言的通知，等将反馈通知给用户）

由于初期投稿的数量很重要，对于优质内容的作者将给予较多的代币。但是随着平台成长，这个比重会做出调整。在成长过程的的不同阶阶段为了使用户投稿和评价的核心驱动力获得最大的效果，将会相应的调整奖励比重相关的参数，但是如果这些调整都是有我们单方面决定和实施将会出现中央集权的不利局面。从而，在适当的时机，我们将通过ALIS代币用户投票来决定如何调整奖励机制的参数。以下表示我们业务初期阶段的系统图。这个图在现阶段仅仅是假设图，在实施阶段将会做进一步的修正和改进。

2. 在所有点评网站领域拓展市场

作为战略的第2阶段而言，我们的平台与饮食，旅行，瘦身等日常领域，或者在入学，结婚，购房等生活领域具有良好的切合性。这是由于人们倾向于在这些领域中通过别人的评价获取值得信赖的信息。例如在日本，yelp日语版的“食べログ”，Retty等服务正在不断成长。这些领域正是ALIS的下一个目标，作为投稿的作者·评述者同时可以获得薪酬这种ALIS提供的独有价值，进入这个市场。

3. 以积累的用户信赖信息为基础，拓展新的服务领域。

关于这一点，从结论而言，我们希望在ALIS上构筑在日本的求职供职系统。作为日本的国策，正在以一亿总活跃社会为基础推进工作制度的改革（http://www.kantei.go.jp/jp/singi/hatarakikata/ ）。简单而言，其重要的措施之一是增加人们的副业机会。这是由于日本社会正在逐渐成为世界上罕见的老龄化社会，受限于严重的劳动人口不足。与此同时每一个劳动者的生产能力并没有得到最大限度的发挥，政府正在探索细化分工，打破企业固有的限制，增加个体工作领域范围的方式提高生产力的方法。“个体的信赖度没有完全没有被可视化”这个课题成为阻挡这项政策的很大的难题。这对这一点，我们认为通过ALIS积累的用户信赖度信息可以发挥重要的作用。我们假设，善于工作的个体的同时也更善于传递信息。更直接而言，善于工作的人得到的信赖也就越多。也就是，将在传播和发掘信信息的同时积累的个体信赖度数据运用到求职领域，和国家政策相辅相成的生态系统具有实现的可能。

如果实现了这种生态系统，为了近一步实现ALIS代币的交换价值和利用价值，钱包功能也在目前的设想中。具体而言就是建立ALIS代币同法定货币的交易所，或者同其他虚拟货币的交易机能。

# 4. 平台介绍和特征

ALIS将给予创造有价值内容作者以及对内容的评价者给予更多ALIS代币。也就是说，这是一个通过报酬奖励体系发掘值得信赖的信息·人的社交媒体平台。由于这种媒体本身的概念比较新颖，我们通过与已经存在的类似服务STEEM比较来说明我们平台具有的特点和优势：

1. 由于我们的代币只有一种，平台使用规则也更简单
2. 积极面对虚拟货币的不安定属性，相比STEEM对货币膨胀率进行更严格的控制，保证平台的长期稳定运营
3. 目前平台的最终的目标是通过“信赖度信息可视化”，与国家政策相呼应，逐步推进是我们的我们的使命

# 6. 为什么可以收集到高品质的内容

让人们发布优质的内容，或者针对其他人的内容进行评价时所必须的驱动力是什么？是对发布优质内容的作者，或积极对内容进行公正评价的评价者给予更多的报酬奖励。同时为他人做出贡献，被认可这种人类的基本需求也是动机之一。我们的首要任务正是设计能够满足人们的种需求的社交媒体平台。ALIS的报酬系统正是针对这种需求赋予强有力的动机。对于作者，自己的文章通过被更多的用户，或者拥有更多ALIS代币的用户评价获得相应的报酬称为可能。针对给予高质量评价的评价者而言，评价者越多的文章，评价时间越早就会得到越多的报酬。除此之外，拥有越多ALIS代币的用户越能够获得更多的ALIS代币。总而言之，拥有更多的ALIS代币意味着创造优质内容或者发掘优质内容能够获得更多的奖励，进而能够进一步增加优秀内容的比率，从而创造一种良性循环。这就是为什么可以ALIS能够收集到高品质的内容。将这种模型以下图所示AARRR为基础的系统图进行分析可以得到同样的结果，可以看出奖励体系是平台成长的关键驱动力。以上虽然仅仅是着眼于奖励的物质面，但是这种奖励同于人获得认可的需求也是密不可分。明示获得奖励的数值，目的是可以直观了解用户对于他人或者平台的贡献都。即使不考虑ALIS代币的交换价值，仅仅是考虑其对用户的被认可价值，类似于Facebook的“赞！xx个”，也许可以期待其可以达到类似的效果。



# 7. 为什么可以给用户回馈价值

我们的代币本身不具有价值。但是随着ALIS平台价值得到认可，ALIS代币就会拥有相应的价值，可以在交易所通过高的价格进行交易。也许您对这一结论的初始印象会抱有怀疑的态度，但是采用同样方案的STEEM已经获得了$3.78亿的估值的事实来看这种方案是具有可行的。

此外，通过STEEM的俄语版GOLOS同样也得到了$10亿以上的估值（2017/07/17）的事实来看，这种方案取得成功的可能性非常高。



# 8. 为什么ALIS可以持续的长期发展 	

ALIS平台的价值能够得到认可有一个最重要的要素，那就是平台具有用户粘性，为用户提供具有依赖性的服务。为此，有各种个方法，其中之一是逐步提升代币价值的机制。我们的代币在平台上有效（本章后续会说明）状态下具有50%的年增值率，这一部分价值奖奖励给对于ALIS平台作出贡献的用户。近年来虚拟货币的目的被限定在交易上，为了规避这种被随时交易的风险而设定了这个数值。拥有越多ALIS代币，同时为平台做出越多努力的用户，就能够获得越多的代币奖励，正是以这种机制为基础，设定了50%的年增值率。但是如果仅是这样，就没有办法预防用户在ALIS代币升值到预期值之后马上进行交易兑换行为。对此，我们根据NEM的PoI（http://nemmanual.net/NEM_Technical_reference_JA/PoI/7_PoI.html ）引入以下代币实效性同时间的逻辑关系式：

```
f(t) = log94(t) (1 <= t < 94)
f(t) = 1 (t >= 1)
```

其中 t 是代币在账户中存储的天数。采用这种逻辑的理由有三点。第一，为了让新用户尽早体验ALIS的魅力，t 越小，代币实效性增长速度就越高。第二，长期用户可以获得ALIS代币的100%价值。第三，每次取出之后，t 从新归零，从而避免了用户可以随意交易行为。真真想为ALIS贡献的用户数越多，这个逻辑就越具有合理性，可以确保平台长期有效的发展。



# 9. 代币是如何生成和发布的

ALIS预定通过ICO筹集资金，初期发行5亿枚，预定将其中的一半交换为Ethereum虚拟货币。发行上限为2.5亿枚，剩下的2.5亿由我们和我们的利益相关方保有。我们保有2.5亿也就是整体上限的50%的理由是，我们能够掌控维持平台健全发展的奖励机制。

但是请注意，当我们拥有最多的代币保有量并不意味着我们拥有平台价值的决定权。同时我们不会随意交易所拥有的代币（我们会通过智能控制系统限制代币的随意交易，具体详情会在今后公布）。

那么，之前有提到通过ICO发布的ALIS代币具有50%的年增值率，那么关于增值的部分的价值如何分配，通过以下2点进行说明。

1. 对被发布优秀内容并被得到认可的作者给予报酬奖励
2. 公认的优秀的文章尽早作出正面评价的用户进行报酬奖励

对于这种奖励机制，ALIS代币的保有量越多，所获得的奖励也会越多。也就是说，长期对于ALIS平台作出贡献，或者ALIS代币保有量多的人们是我们最重要的用户群里，优先考虑这些最主要用户的利益是我们的规则之一。这个和PoI机制类似，如果平台的健全性遭到破坏，有越多受到负面影响的重要用户，那么这些用户就会对于平台的健全性做出贡献。此外，如果平台上突然出现通过购买ALIS代币的垄断用户，就有可能会左右平台的价值以迎合自己的意愿。

针对这个风险，正如前面所述，ALIS代币的时效性在100%效需之前有一个逐渐解冻的过程。防止保有大量代币用户在短时间内进行大额交易的同时，有能保证代币在平台上被持续保有时间，可谓一举两得。



# 10. 关于详细的发布逻辑和参数调整 

首先先介绍ALIS代币的整体发行量。正如前面提到的ALIS代币的预定发行数量的上限为5亿枚，其中最高2.5亿将在Crowdsale销售。我们的团队将回收销售剩余的代币，并将其在3年间冻结。在通过Crowdsale发布ALIS代币中，假设放入exchange wallet中的比例为X1，放入ALIS wallet中的比例为X2，对于X2引入50%的年增值率。这部分通过增值获得的代币将通过奖励机制分配给投稿作者和评价者。在详细介绍分配逻辑之前，首先先介绍我们的基本原则。

1. 文章的投稿作者和评价者都应该被得到重视和尊重，但是因为作者相对会付出更多的劳动，所以会加重针对作者的奖励份额
2. 文章如果获得的点赞数越多，作者获得的代币奖励也就越多，同样，评论如果获得的点赞数越多，评论者获得的代币奖励也就越多
3. 代币的奖励量，奖励逻辑会随着平台的发展不断得到进化改善

通过上述的原则，我们会将ALIS代币中的90%分配给作者，10%分配给评价者（假设达成我们ICO最低集资额3.5亿日元的情况下，在第一年对于作者使用6700万日元，评价者使用750万日元进行奖励分配。注意，这个是在ALIS价值不发生任何改变的假设下）

这是由于平台初期投稿的数量非常重要。但是，随着投稿数量的逐渐上升，针对投稿内容评价的重要度也会逐渐上身。在这个阶段针给予评价者的奖励应该会有所提高。这就需要在平台发展的不同阶段调整奖励分配规则的系数，如果这种调整方案全部有我们运营者自己决定，就会产生中央集权的不利局面。所以，当用户希望通过交涉参与的方式进行系数调整时，会进行以ALIS代币所有量作为权重的民主投票。我们正在考虑将这种，根据总体用户意愿（51%以上同意）调整相关系数，的规则作为我们的未来平台运营方案。

# 11. 如何防止不正当操作 

根据上述的分配逻辑，如果允许以下不正当的方式取得代币，就有可能损害大多数用户的利益。

1. 通过同时注册多个账号，对自己的文章进行大量评价来获得ALIS代币奖励。
2. 通过和特定的用户合作，对制定的文章进行大量评价来获得ALIS代币奖励。

我们已经准备了防止这些不正当行为的对策。首先针对第1点，账号无法被轻易的注册。具体而言，比如在注册时需要通过短信进行本人确认，或者必须通过Facebook认证登录。针对第2点，

1. 当特定用户在短时间内评价大量文章时进行无效化设定（因为在短时间内能够同时阅读大量文章的用户，统计学上应该非常少）
2. 通过文章的负面评价的比例来分析给予负面评价用户的行为，如果确认为不正行为，将给予其的奖励无效化。

等对策来处理不正行为。这受启发于STEEM已经采用的不正行为防止规则。这在STEEM平台上能够被有效运用，相信在我们的平台上也会同样有效。


# 12. 关于ALIS所使用的区块链技术


值得肯定的是，用于可视化人们信赖度的最合适的技术，莫过于ALIS核心，区块链技术。

在要求最高标准可靠性的通货·结算领域，正如被实现的去中心化的比特币为代表而言，对于可靠性这一点而言区块链技术相比较于现存的技术而言具有压倒性的优势，甚至可以说不在同一个比较层次。

比特币所实现模式可以说是一种艺术，我们感受到正处于前人未曾到达的社会变革之时。这正是我们创立ALIS的原动力。至少目前而言，在日本没有在区块链技术之上构筑人们信用情报的先例。所以一旦构筑了这个系统，可以确信在日本更容易获得无法被颠覆的压倒性优势。没有比这个更合适的形式实现，如 Peter Andreas Thiel 所描述的成功企业必须满足的条件之一"独占性"，

我们将区块链技术用于保存评价信息和报酬数据。这个保存操作通过将ALIS币放入账户中进行激活的用户进行进行授权。这些用户判断前述点数的合理性之后，对各个用户进行合理分配。这样，在最容易出现不正当操作的虚拟货币发行的部分使用了与之非常匹配的区块链技术。

另外，我们现在正在探讨在Ethereum，Microsoft Azure（我们现在作为Azure BizSpark合作伙伴，通过的微软公司的认证，同时也正在申请BizSpark+）等技术中，最值得信赖，同时低成本的实现方案。由于本章所提及的技术问题的进一步探讨还未完成，后续如果有追加的信息，我们会进行同步更新。

目前，预定在评价和报酬部分使用区块链技术，同时也正在探讨其他功能所采用的相关技术。

# 16. 企业运营方针

我们感觉到公司股份制已经非常过时。首先股份制有很多不协调的地方。公司虽然被说成是股东的，实际上股东所能得知信息仅仅是通过股东大会被修饰的信息。除此之外，具有经营权的董事只有在认为自己绝对正确的前提下才会下决定，同时也并没有顾及现场职员的想法。正如Gary Hamel 的《未来经营》一书中所提到的，公司系统是在数百年的时间停止进化的活化石，针对这种情况我们希望按照以下的方针进行公司运营。

1. 通过trello等工具将项目进展情况全部透明化
2. 将成员间的交流也尽可能的公开化
3. 将源代码全部作为开源项目在github公开
4. 公司未来的方向以及基本规则修正都通过代币所有人进行民主投票方式确定
5. 员工工资在员工之间全部公开

加上我们的所有费用都是来自每一位投资者。对着投资者，尽可能的透明化企业经营是我们的义务，即使按照这种里面，上述的方针也是值得尝试和挑战的。

# 17. 为什么在香港实施ICO 

我们在日本开发平台，却在香港发布ICO。我们发现很对人表现出不理解和不安。其实理由很明确，在日本2017年4月之后随着《改正资金决济法》（http://www.fsa.go.jp/common/about/20170403.pdf）的实施，禁止了日本国内公司实施ICO。我们当初也是预定在日本ICO，但是由于最近国会的讨论，在日本ICO有很高的可能违反法律，于是决定在海外实施ICO。关于选择香港的理由是，是创立和维持公司的成本考虑，同时香港并没有关于虚拟货币的限制法律。在香港发行代币的基础上，在日本申请公司，从而使无论是日本还是其他国家的用户都能够没有屏障的进行代币交易是我们的目标。此外，通过成为日本国内公司的另一个非常重要的优点就是就可以在日本的虚拟货币交易所进行代币交易。我们已经和日本国内最大的虚拟货币交易所进行了接触，正在探讨未来上市的可能性和方法。请注意在我们申请成为日本公司之前，只能在海外交易所进行代币交易。另外，关于香港法人将来仅仅是作为ICO的目的存在，但是由于日本法律和其他问题，有日本法人运营出现困难时，将作为备份的存在目的，预定将长期维持下去。
